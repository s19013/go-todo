# createTodo
todoを新しく作る
## Request
* Subject
* Description

### バリデーション
Subjectは空文字ではだめ

## Response
* Id
* Message

#### Id
送信後に編集するときに使う

# getTodo
todoを単独取得  

method:get  
url:/todo/(id)  

### バリデーション
idは必須,空文字ng

## request 
* Id

## response
* todo

#### todo
todoのデータ
